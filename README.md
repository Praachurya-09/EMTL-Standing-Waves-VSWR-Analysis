## Standing Waves & VSWR Analysis

## Overview

This project explains and analyzes **Standing Waves** and **Voltage Standing Wave Ratio (VSWR)** in transmission lines using both **theory and simulation**.

---

## What are Standing Waves?

Standing waves are formed when an incident wave reflects back due to impedance mismatch, creating a pattern of nodes and antinodes.

---

## What is VSWR?

VSWR (Voltage Standing Wave Ratio) is defined as:

VSWR = Vmax / Vmin

* VSWR = 1 → Perfect matching
* VSWR > 1 → Impedance mismatch

---

## Reflection Coefficient (Γ)

Γ = (ZL - Z0) / (ZL + Z0)

---

## Python Simulation

This project uses Python to:

* Simulate incident and reflected waves
* Generate standing wave pattern
* Calculate Vmax, Vmin, and VSWR
